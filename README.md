# Fruity UI

A small React + Vite + Tailwind demo app inspired by the provided mobile screens.

Quick start

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Open the browser at the Vite dev URL (usually http://localhost:5173).

Features

- React Router for separate screens (Home / Cart / Profile)
- Persistent cart stored in `localStorage`
- Mobile app preview container sized to 375x812

Notes

- This is a minimal scaffold to preview the UI. Extend components and add backend integration as needed.
